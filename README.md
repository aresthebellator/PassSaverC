# PassSaverC
PassSaverC è un software scritto puramente in C, il programma è perfetto per chi ha bisogno di un password manager che non consumi RAM, tutto grazie alla sua interfaccia CLI. 
Il programma è stato pensato per chi sta facendo girare in locale una LLM è quindi ha bisogno di risparmiare il più possibile nella RAM, o per chi ha un computer abbastanza lento e vuole preservare il più possibile le prestazioni.

##PER INSTALLARLO E COMPILARLO:
 
```bash
git clone [https://github.com/aresthebellator/PassSaverC.git](https://github.com/aresthebellator/PassSaverC.git)
cd PassSaverC
gcc PassSaverC -o PassSaverC
./PassSaverC
